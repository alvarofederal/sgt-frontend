To-Do-List Gerencie tudo com facilidade

Olá! Sistema FullStack, utlizando Java, com SpringBoot, no Backend, e Angular no Frontend.
Esse é um sistema para gerenciamento de tarefas, desenvolvido em Angular 17 e Spring Boot 2.
Esse projeto consiste em gerenciar demandas, no modelo To-Do-List, onde, você cadastra as tarefas, e define a validade.
O sistema ainda conta com a Criação, Edição, Listagem e Exclusão de Tarefas, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte deste projeto pode ser acessado clicando no link Github no menu ao lado.

Foi um desafio interessante, desenvolver um sistema completo tão funcional e em tão pouco tempo.
